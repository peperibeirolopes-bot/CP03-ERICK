## Descrição do sistema

O projeto consiste em um simulador de caixa eletrônico (ATM) desenvolvido em linguagem C, executado via terminal. O sistema permite que o usuário realize operações bancárias básicas por meio de uma interface textual organizada, simulando o funcionamento de um caixa eletrônico real. Foram utilizados conceitos de lógica de programação, estruturas de decisão, repetição e funções.

## Funcionalidades implementadas

- Sistema de login com número da conta e senha;
- Limite de 3 tentativas de acesso;
- Consulta de saldo disponível;
- Depósito de valores;
- Saque com validação de saldo;
- Alteração de senha;
- Menu interativo via terminal;
- Validação de entradas inválidas;
- Encerramento do sistema.
